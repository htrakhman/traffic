# Outline

## Hook / AEO answer span (first 100 words)

> Short answer: **Type I and Type II barricades are temporary
> channelizers** — Type I is one rail, Type II is two rails, both used
> for short-duration day-time work and pedestrian routing. **Type III is
> the heavy-duty, full-roadway closure barricade** — three reflective
> rails on a sturdy frame, required by MUTCD for road closures, long-term
> work zones, and any setup expected to be hit by traffic. If you're
> closing a road, detouring traffic, or working overnight on a high-speed
> route, you need Type III. For a daytime sidewalk repair, Type I or II
> is fine.

## H2: Type I — the "minimum acceptable" barricade

- Single horizontal reflective rail, ~24" wide
- Lightweight (often plastic, sometimes metal)
- Used to outline pedestrian paths, short utility holes, side-of-road
  digs that are barely a hazard
- Limit: not allowed for closures of any travel lane or shoulder on
  high-speed roads (MUTCD restriction)
- Best for: sidewalk closures, parking-lot work, festival/event routing
- When NJ municipalities accept Type I: low-speed local streets,
  daytime only, with proper advance signage

## H2: Type II — when one rail isn't enough

- Two horizontal rails (top + bottom)
- Same frame footprint as Type I but with the added second rail for
  drivers/peds approaching from a steeper angle
- Used for: short-duration shoulder work, lane shifts on low-speed roads,
  channelizing pedestrians around larger excavations
- The "default" for most utility-cut and small-pothole work where
  Type I feels too flimsy

## H2: Type III — the road closure standard

- Three full-width reflective rails on a heavy frame (often weighted
  with sandbags)
- Required by MUTCD for:
  - **Full road closures** ("ROAD CLOSED" mounted to the barricade)
  - **Detour routes** (mounted detour signs)
  - **Long-term work zones** (more than 3 days, per most state DOTs)
  - **Nighttime work on roads ≥40 mph** where lane is taken
  - **Critical work zones** (active travel lane, no taper protection)
- Visibility: visible from 1,000+ ft at night with high-intensity
  prismatic sheeting
- Often paired with Type 1 warning lights (steady-burn) at corners

## H2: How to pick — the decision tree

A short decision flow contractors can use on-site:

1. **Are you closing a travel lane or road?** → Type III.
2. **Is the work zone live overnight?** → Type III.
3. **Speed limit at the work zone ≥40 mph?** → Type III if the barricade
   is the closure marker; Type II if it's just a channelizer.
4. **Pedestrian route only, daytime, under-35 mph street?** → Type I or II.
5. **Confused?** Default to Type II; you'll never get cited for over-spec'ing.

## H2: What NJ adds on top of MUTCD

- NJDOT Standard Specifications Section 159 reference (verify current
  edition at draft time)
- Municipal permit nuance: many NJ towns require Type III for any
  overnight road work regardless of speed
- NJ Turnpike / GSP rules: Type III plus "ROAD CLOSED" placard, plus
  approved arrow board, plus shadow vehicle for any travel-lane closure
- Pedestrian routing in NJ: ADA-compliant detection (cane-detectable
  bottom rail) — a detail Type I sometimes fails

## H2: Common mistakes contractors make

- **Using Type I for a road closure** — the cite-magnet
- **Skipping ballast on Type III** — wind takes them out within an hour
  on highway work
- **Mixing rail directions** — reflective sheeting must face oncoming
  traffic, both sides if barricade is between two flow directions
- **Letting damaged barricades back into rotation** — bent rails fail
  the MUTCD reflectivity test even if the sheeting looks fine

## H2: Rent vs buy — which one makes sense?

- Type I and II: cheap enough that owning is fine if you do this work
  weekly (sub-$100 each)
- Type III: $400–800 retail. Rent unless you have a multi-week closure
  job or you do 1+ closures per month
- Storage cost: Type IIIs take real yard space and get damaged in
  storage. Renting moves that risk to the supplier.

## FAQ (FAQPage schema, 5–6 Q/A)

1. **What's the difference between Type I, Type II, and Type III barricades?** —
   Number of rails (1 / 2 / 3), height (36 in / 36 in / 60 in), and use
   case (pedestrian/short-day / shoulder work / road closures + nighttime).
2. **Can I use a Type II barricade to close a road?** — No. MUTCD requires
   Type III for any full road closure or detour route.
3. **Do I need a Type III barricade for nighttime utility work?** — If
   you're taking a travel lane or working on a road ≥40 mph at night, yes.
   For a parking-lot or sidewalk job, Type I/II is fine.
4. **How much does a Type III barricade rental cost in NJ?** — Typically
   $25–50/day per barricade; verify with quote (high-intensity sheeting
   and weighted bases sit at the top of that range).
5. **What weight ballast should a Type III barricade have?** — At minimum
   25 lb sandbags on each leg; for highway-speed work zones, double that
   or use water-filled bases.
6. **Are plastic Type III barricades MUTCD-compliant?** — Yes, if they
   meet NCHRP-350 or MASH crashworthiness and the reflective sheeting
   meets ASTM Type IV/IX. Cheap plastic units from non-compliant
   suppliers will fail an inspection.
