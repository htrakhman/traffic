# Lead Tracker

*Source of truth for "who has been contacted." Updated by scheduled
tasks and manual chat sessions.*

## How this file is structured

Each entry below corresponds to a first-touch outreach. Don't delete
entries — just append status updates. Format:

```
### [Company Name] — [County]
- **Added to queue:** YYYY-MM-DD
- **Template:** A / B / C
- **Sent via:** FB Messenger / Email
- **Sent on:** YYYY-MM-DD
- **Response:** None / Hot / Warm / Cold / DNC
- **Latest note:** free text
- **FB URL:** ...
- **Website:** ...
```

Status terms:
- **None** — no reply yet
- **Hot** — replied with real interest or asked for pricing
- **Warm** — replied but noncommittal
- **Cold** — replied dismissively
- **DNC** — "do not contact" / hostile — respect and remove from future pulls
